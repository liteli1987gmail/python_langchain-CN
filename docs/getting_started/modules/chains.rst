链
==========================



注：
 `概念指南 <https://docs.langchain.com/docs/components/chains>`_




在某些简单的应用程序中，单独使用LLM是可以的，
但更复杂的应用程序需要将LLM链接起来-可能是彼此链接，也可能是与其他专家链接。
LangChain提供了标准的**链**接口，以及几种常见的链实现。
