犬类寻回犬

==========================



注意：

   `概念指南 <https://docs.langchain.com/docs/components/indexing/retriever>`_





寻回器接口是一种通用接口，使文档与语言模型组合变得容易。该接口公开了一个 `get_relevant_documents` 方法，该方法接受一个查询

（一个字符串）并返回一个文档列表。
