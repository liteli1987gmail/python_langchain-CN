# 亚马逊 Bedrock


>[亚马逊Bedrock]（https://aws.amazon.com/bedrock/）是一个全面托管的服务，通过API使来自领先AI创业公司和亚马逊的FMs可用，因此您可以从广泛的FMs中选择最适合您用例的模型。


## 安装和设置


```bash
pip install boto3

```
