# Amazon Bedrock 亚马逊基岩


>[Amazon Bedrock](https://aws.amazon.com/bedrock/)是一个完全托管的服务，通过API提供来自领先人工智能初创公司和亚马逊的模型，因此您可以从各种FMs中选择找到最适合您的用例的模型。


## 安装和设置


```bash

pip install boto3
```
