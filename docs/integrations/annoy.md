# Annoy 恼人


> [Annoy](https://github.com/spotify/annoy) (`Approximate Nearest Neighbors Oh Yeah`) 是一个带有Python绑定的C++库，用于搜索与给定查询点接近的空间点。它还创建大型的只读基于文件的数据结构，这些结构被映射到内存中，以便许多进程可以共享相同的数据。

## 安装和设置





```bash

pip install annoy

```
