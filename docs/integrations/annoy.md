注释



> Annoy](https://github.com/spotify/annoy)（近似最近邻搜索）是一个C++库，具有Python绑定，用于搜索与给定查询点接近的空间点。它还创建了大型的只读基于文件的数据结构，将其映射到内存中，以便多个进程可以共享相同的数据。

安装和设置





```bash

pip install annoy

```
