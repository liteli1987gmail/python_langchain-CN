# Annoy（近似最近邻搜索）


> [Annoy](https://github.com/spotify/annoy) 是一个 C++ 库，带有 Python 绑定，用于搜索与给定查询点接近的空间点。它还创建大型只读基于文件的数据结构，这些结构在内存中被映射，因此许多进程可以共享相同的数据。
## 安装和设置




```bash

pip install annoy

```
