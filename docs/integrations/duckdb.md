DuckDB(水鸭DB)


> [DuckDB](https://duckdb.org/) 是一个基于进程的 SQL OLAP 数据库管理系统。


## 安装和设置


首先，您需要安装 `duckdb` Python 包。


```bash
pip install duckdb

```
