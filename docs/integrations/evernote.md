## 印象笔记


>[印象笔记](https://evernote.com/)是用于存档和创建笔记的应用程序，其中可以嵌入照片、音频和保存的网络内容。笔记存储在虚拟的“笔记本”中，可以进行标记、注释、编辑、搜索和导出。


## 安装和设置


首先，您需要安装`lxml`和`html2text` Python软件包。


```bash

pip install lxml

pip install html2text

```
