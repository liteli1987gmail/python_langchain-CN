>Google BigQuery](https://cloud.google.com/bigquery)是一个无服务器、成本效益高的企业数据仓库，可以跨云平台工作并随着数据规模扩展。
`BigQuery`是`Google Cloud Platform`的一部分。

安装和设置

首先，您需要安装`google-cloud-bigquery` Python包。

```bash

pip install google-cloud-bigquery
```
