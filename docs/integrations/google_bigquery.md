# Google BigQuery



>[Google BigQuery](https://cloud.google.com/bigquery)是一个无服务器且具有成本效益的企业级数据仓库，适用于各个云端且与数据一起扩展。

`BigQuery` 是 `Google Cloud Platform` 的一部分。



## 安装和设置



首先，您需要安装 `google-cloud-bigquery` python 包。



```bash

pip install google-cloud-bigquery

```
