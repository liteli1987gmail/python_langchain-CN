# Google Drive 谷歌驱动器


>[Google Drive](https://en.wikipedia.org/wiki/Google_Drive) 是由谷歌开发的一个文件存储和同步服务。



当前只支持 `Google Docs`。



## 安装和设置



首先，您需要安装几个 Python 包。



```bash

pip install google-api-python-client google-auth-httplib2 google-auth-oauthlib

```
