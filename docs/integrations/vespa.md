Vespa（音译为“唯思帕”）


>[Vespa](https://vespa.ai/)是一个功能完备的搜索引擎和向量数据库。
>它支持向量搜索（ANN）,词汇搜索,以及在同一查询中搜索结构化数据,。
 

##安装和设置




```bash

pip install pyvespa

```
