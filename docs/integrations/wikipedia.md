# 维基百科


>[Wikipedia](https://wikipedia.org/)是一个由志愿者社群,即维基人,通过开放协作和使用名为MediaWiki的wiki编辑系统撰写和维护的多语种免费在线百科全书。 `维基百科`是历史上最大的和最受欢迎的参考工具。




## 安装和设置


```bash
pip install wikipedia

```
