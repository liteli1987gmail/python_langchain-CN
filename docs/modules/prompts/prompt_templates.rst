提示模板
==========================



注意：
   `概念指南 <https://docs.langchain.com/docs/components/prompts/prompt-template>`_





语言模型接受文本作为输入 - 这个文本通常被称为提示。
通常这不仅仅是一个硬编码的字符串，而是一个模板、一些例子和用户输入的组合。
LangChain 提供了几个类和函数，使得构建和使用提示变得简单。


提供以下文档部分：


- `入门指南 <./prompt_templates/getting_started.html>`_：LangChain 用于处理和构建提示的所有功能概述。


- `操作指南 <./prompt_templates/how_to_guides.html>`_：一个操作指南的集合，这些指南强调了如何使用我们的提示类完成各种目标。


- `参考资料 <../../reference/prompts.html>`_：所有提示类的 API 参考文档。
