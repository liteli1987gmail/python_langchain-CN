## 链


如果您刚刚开始，且您有相对简单的API，您应该从链开始。

链是一系列预定步骤，所以它们很适合刚刚开始，因为它们给您更多的控制权，让您更好地了解正在发生的事情。

代理的缺点是您的控制力较小。优点是它们更强大，这使您可以在更大更复杂的模式上使用它们。
